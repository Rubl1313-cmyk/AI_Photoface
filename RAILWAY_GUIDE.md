## 🛠️ Развертывание на Railway.app

### 1. Подготовка проекта:
```bash
# Клонируй репозиторий
git clone <your-repo>
cd AI_Photoface

# Переименуй файлы для Railway
cp railway_main.py main.py
cp railway_requirements.txt requirements.txt
cp Railway-Dockerfile Dockerfile
```

### 2. Переменные окружения в Railway:
```env
# Telegram Bot
TG_BOT_TOKEN=your_bot_token
ADMIN_ID=your_admin_id
PORT=8000

# Cloudflare Workers AI
CF_WORKER_URL=https://your-worker.workers.dev

# FaceFusion API
FACEFUSION_URL=https://your-hf-space.hf.space

# Лимиты
DAILY_LIMIT=50
```

### 3. Развертывание:
```bash
# Установи Railway CLI
npm install -g @railway/cli

# Войди в Railway
railway login

# Создай новый проект
railway init

# Разверни
railway up
```

### 4. Настройка вебхука:
```bash
# Получи URL Railway
railway domain

# Установи вебхук для Telegram
curl -X POST "https://api.telegram.org/bot<TG_BOT_TOKEN>/setWebhook" \
     -H "Content-Type: application/json" \
     -d '{"url": "https://<your-railway-url>.railway.app/webhook"}'
```
