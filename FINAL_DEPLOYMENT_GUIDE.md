## 📋 **Пошаговый план развертывания:**

### 🤗 **Шаг 1: Разверни HuggingFace FaceFusion**
1. Создай Space: [huggingface.co/spaces](https://huggingface.co/spaces)
2. Выбери "Docker" template
3. Загрузи файлы из `huggingface-space/`
4. Получи URL: `https://your-space.hf.space`

### ⚡ **Шаг 2: Обнови Cloudflare Worker**
```bash
cd cloudflare-worker
wrangler deploy
```
Получи URL: `https://your-worker.workers.dev`

### 🚀 **Шаг 3: Разверни бот на Railway**
1. Создай проект на [railway.app](https://railway.app)
2. Добавь переменные окружения:
   ```
   TG_BOT_TOKEN=твой_токен
   CF_WORKER_URL=https://your-worker.workers.dev
   FACEFUSION_URL=https://your-space.hf.space
   PORT=8000
   ```
3. Разверни проект
